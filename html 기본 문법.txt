1. head 태그, body 태그

    - head 태그 : 문서의 특성, 정보를 알려주는 태그

    - body 태그 : 문서 본문을 표시하는 태그
2. tltle 태그

    - 웹페이지의 제목을 표시해주는 태그

    - 검색 엔진들은 tiltle 태그를 책 표지처럼 인식한다. (검색 결과 노출을 위해 잘 지정해야할 듯)

​

3. meta 태그

    - 웹 서버와 웹 브라우저간에 상호 교환되는 정보를 정의할 때 사용

    - 문서 자체의 특성을 담는 태그

​

4. a 태그

    - URL 링크를 만드는 태그

    - Hypertext를 상징

    - <a href = 'URL 주소'>

​

5. 기타

    - strong 태그 : 글씨 굵게

    - u 태그 : 밑줄

    - h 태그 (h1~h6) : h1부터 h6까지 있고, 숫자가 커질수록 글자 사이즈가 작아진다.

    - br 태그 : 줄바꿈

    - p 태그 : 단락 지정

    - img 태그 : 이미지 삽입 <img src = '이미지 경로'> 

    - ul(unordered list) 태그 / ol(ordered list) 태그 / li 태그: 

       * <oi><li> 내용 </li></oi> : 숫자로 리스트를 보여준다. 

       * <ui><li> 내용 </li></ui> : 숫자 없이 점으로 리스트를 보여준다.

    - hr 태그 : 수평 가로선

    - span, div 태그 : 웹페이지에 레이아웃을 구성하고 영역을 설정할 때 사용 / div > span
